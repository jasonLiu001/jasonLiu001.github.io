 ## 使用心得

#### 几个概念

1. 选区
2. 图层
3. 蒙版
4. 通道

## 操作技巧

+ 拷贝选区内容并生成新的图层

  ```shell
  Ctrl-J
  ```

+  拷贝所有可见图层中的选区部分

  ```shell
  # 拷贝选区部分
  Ctrl-Shift-C
  # 粘贴的同时会生成新图层
  Ctrl-V
  ```

+ 拷贝所有可见图层同时合并后生成新的图层

  ```shell
  Ctrl-Alt-Shift-E
  ```

+ 按住`Ctrl`后点击对应图层，可以生成当前图层的选区

+ 撤销对图层的修改

  ```shell
  Ctrl-Alt-Z
  ```

+ 使用套索工具时，选取到屏幕边缘时，按住空格键可以变成小手状，以便来拖到图片
